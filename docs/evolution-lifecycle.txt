  VALIDATOR EPOCH LOOP (every 60s)
  ────────────────────────────────

    1. Resync metagraph
    2. Register miner manifests (node discovery)
    3. Challenge miners on 4 axes:
       ├── Corpus integrity  (Merkle proof verification)
       ├── Traversal         (embedding similarity + latency)
       ├── Quality           (narrative passage + choice fairness)
       └── Topology          (betweenness centrality + edge weight)
    4. Aggregate via EmissionCalculator:
       ├── Traversal pool  40%  (linear normalization)
       ├── Quality pool    35%  (softmax)
       └── Topology pool   25%  (rank normalization)
    5. Set weights (Yuma Consensus)
    6. Decay edges (multiplicative, half-life ~1.2 days)
    7. Pruning: push scores + process collapses (every 10 epochs)
    8. Voting: finalize expired proposals + return/burn bonds
    9. Integration: advance ramp-in nodes + register newly-live
